# Zipcode Search Application

## Overview
This application provides a simple user interface for searching geographical details based on zip codes. It interacts with an external API to fetch and display information such as city, state, country code, latitude, and longitude.

## Technologies Used
- Java with Spring Boot for the backend
- HTML and CSS for the frontend
- External Zipcode API for data

## Running the Application
1. Ensure you have Java and Spring Boot installed.
2. Start the backend server by running the Spring Boot application.
3. Open the `index.html` file in a web browser to access the frontend.
4. Enter a zipcode and click the "Search" button to retrieve and display data.

## Features
- Simple and intuitive interface for zipcode search.
- Dynamic display of search results.
- Error handling for invalid or non-existent zipcodes.
